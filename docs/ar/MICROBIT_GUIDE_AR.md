# 🔌 رفيق micro:bit — دليل الإعداد

## ما تحتاجه

- BBC micro:bit V2 (V1 يعمل لكن بدون مكبر صوت)
- كابل USB أو حافظة بطاريات
- متصفح Chrome أو Edge (لـ Web Bluetooth)
- تطبيق طائرات في السماء مفتوح

## البدء السريع (دقيقتان)

### الخطوة 1: تفليش micro:bit

**الخيار أ — MakeCode (الأسهل، 6 سنوات فأكثر)**
1. اذهب إلى [makecode.microbit.org](https://makecode.microbit.org)
2. مشروع جديد ← تبويب **JavaScript**
3. **الإضافات** ← ابحث عن **bluetooth** ← أضفه
4. الصق محتوى `microbit_makecode.js`
5. **تنزيل** ← اسحب ملف .hex إلى محرك micro:bit

**الخيار ب — MicroPython (10 سنوات فأكثر)**
1. اذهب إلى [python.microbit.org](https://python.microbit.org)
2. الصق محتوى `microbit_planes.py`
3. اضغط **إرسال إلى micro:bit**

### الخطوة 2: الإقران مع التطبيق
1. افتح طائرات في السماء في Chrome/Edge
2. اضغط زر **🔌** (أعلى اليمين)
3. اختر micro:bit من نافذة البلوتوث
4. نبضة خضراء = متصل!

## أزرار التحكم

| الإجراء | ما يفعله |
|---------|----------|
| **الزر A** | تغيير وضع العرض: رادار ← عدّاد ← بوصلة ← علم |
| **الزر B** | التنقل بين الطائرات، تمرير المعرّف والمسافة |
| **الهز** | طلب بيانات جديدة من التطبيق |

## أوضاع العرض

### 🟢 رادار (الافتراضي)
شبكة LED 5×5 تصبح شاشة رادار مصغّرة:
- **النقطة المركزية** = موقعك
- **النقاط المحيطة** = طائرات قريبة
- أسطع = ارتفاع أعلى
- الموقع يطابق السمت الحقيقي بالبوصلة

### 🔢 العدّاد
يعرض العدد الإجمالي للطائرات كرقم كبير. يتحدث كل 5 ثوانٍ.

### 🧭 البوصلة
سهم يشير نحو أقرب طائرة. يستخدم بوصلة ومقياس تسارع micro:bit المدمجين. أمِل وأدر micro:bit لـ «تجد» الطائرة في السماء!

### 🏳️ العلم
يعرض رمز بلد أقرب طائرة بالتمرير على LEDs.

## التنبيهات 🚨

عند مرور طائرة على بُعد أقل من 8 كم:
1. رسوم حلقة متوسعة على LEDs
2. أيقونة طائرة تظهر
3. الجرس يعزف وتراً صاعداً C-E-G (V2 فقط)
4. المعرّف والمسافة يمرّان على الشاشة

## بروتوكول البيانات

يرسل التطبيق البيانات عبر BLE UART كل 5 ثوانٍ:

```
U|العدد|معرّف,مسافة,سمت,ارتفاع,بلد|معرّف,مسافة,سمت,ارتفاع,بلد|...
```

مثال:
```
U|14|AFR123,42.5,210,10668,FR|BAW456,8.1,45,8534,GB|UAE789,95.3,180,11278,AE
```

رسائل التنبيه:
```
A|BAW456|5.2|8500|GB
```

يمكن لـ micro:bit الرد بـ:
```
REFRESH
```

## حل المشاكل

**« Web Bluetooth غير مدعوم »**
→ استخدم Chrome أو Edge. Safari و Firefox لا يدعمان Web Bluetooth بعد.

**micro:bit لا يظهر**
→ تأكد أن البلوتوث مفعّل في الكود (MakeCode: أضف إضافة bluetooth)
→ أعد تشغيل micro:bit
→ ألغِ الإقران من إعدادات البلوتوث إذا كان مقترناً مسبقاً

**لا بيانات على LEDs**
→ تحقق من النبضة الخضراء على زر 🔌
→ تأكد أن الطائرات محمّلة في التطبيق أولاً
→ اضغط الزر A للتأكد أنك في وضع الرادار

**البوصلة تشير خطأ**
→ عايِر: عندما يعرض micro:bit « TILT »، أمِله في كل الاتجاهات حتى تمتلئ الشاشة

## أفكار مشاريع

- 📊 **مسجّل بيانات**: أضف استدعاءات `datalogger.log()` لتتبع الطائرات عبر الزمن
- 🔊 **نغمات الارتفاع**: اربط ارتفاع الطائرات بنوتات موسيقية
- 🌈 **شريط NeoPixel**: أضف شريط LED لتصوّر كثافة حركة المرور
- 📻 **شبكة راديو**: عدة micro:bit تتشارك بيانات الطائرات عبر الراديو
- 🧪 **مشروع علمي**: رسم بياني لعدد الطائرات بالساعة/اليوم وربطه بالطقس
